```python

x.tofile(fname)
```

```python
import matplotlib.pyplot as plt
>>> N = 8
>>> y = np.zeros(N)
>>> x1 = np.linspace(0, 10, N, endpoint=True) # start -> stop 까지의 N개의 point를 갖는 행렬 선언
>>> x1
>>> x2 = np.linspace(0, 10, N, endpoint=False) # 위의 경우 stop의 값을 포함하지 않음
>>> x2
>>> plt.plot(x1, y, 'o')
>>> plt.plot(x2, y + 0.5, 'o')
>>> plt.ylim([-0.5, 1]) # plot의 가시성을 위해 범위 설정
>>> plt.show()
```
